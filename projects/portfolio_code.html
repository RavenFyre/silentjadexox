Portfolio Web Page

Overview

This portfolio has been made in HTML with CSS and Javascript, and uses a publicly-available Jekyll theme for the design. It has been created in the last week to highlight some of my recent and older projects, and demonstrate my writing, programming and web-development capabilities, purely for the purpose for the FIS Development Unit to review.

Features

  • HTML & CSS creation coded from the ground up.
  • Use of Javascript to enhance interactivity, such as button highlighting on-hover, and images that fade-in in the image carousels.
  • Three of the images used in this portfolio, displayed below, have been put together by Jade using Gimp (original images taken from the following online generators; Jade's avatar is from Habbo Imager and the furniture is from Habbo Furni).
